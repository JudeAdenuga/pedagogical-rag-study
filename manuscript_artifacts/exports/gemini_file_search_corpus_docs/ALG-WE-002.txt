Title: Every term inside parentheses must receive the multiplier

Instructional content:
When using the distributive property, the multiplier applies to every term inside the parentheses. For 3(x + 4), multiply 3 by x and also multiply 3 by 4. The result is 3x + 12, not 3x + 4.
